# Contribuer à H.A.C.A

Merci de votre intérêt pour contribuer !

Veuillez signaler les problèmes et soumettre des "pull requests" sur le [dépôt GitHub](https://github.com/JeanMarc-Labs/ha-config-auditor).
1. Créez un fork du projet.
2. Créez votre branche de fonctionnalité (`git checkout -b feature/super-fonction`).
3. Validez vos changements.
4. Poussez sur la branche.
5. Ouvrez une Pull Request.
